# Portfolio Premium em React

Portfolio pessoal premium feito com React, TypeScript, Tailwind CSS, Framer Motion e componentes reutilizaveis prontos para GitHub Pages.

## Como usar

1. Instala as dependencias com `npm install`
2. Corre localmente com `npm run dev`
3. Faz build com `npm run build`
4. Publica no GitHub Pages com `npm run deploy`

## Antes de publicar

- Atualiza o campo `homepage` em [package.json](/Users/goncalo.santos/Desktop/portfolio/package.json:5) com o teu utilizador do GitHub.
- Se o nome do repositorio mudar, atualiza tambem `base` em [vite.config.js](/Users/goncalo.santos/Desktop/portfolio/vite.config.js:5).
- Personaliza os teus dados em [src/data/site.ts](/Users/goncalo.santos/Desktop/portfolio/src/data/site.ts:1), [src/data/projects.ts](/Users/goncalo.santos/Desktop/portfolio/src/data/projects.ts:1), [src/data/process.ts](/Users/goncalo.santos/Desktop/portfolio/src/data/process.ts:1) e [src/data/skills.ts](/Users/goncalo.santos/Desktop/portfolio/src/data/skills.ts:1).
